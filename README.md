## 🚀 Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/KOSALSENSOK096/MyPortfolio.git
   cd MyPortfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```
